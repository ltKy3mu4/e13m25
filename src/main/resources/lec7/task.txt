Создать Абстрактный класс Животное с абстрактным методом идти
Создать 2 реализации Животное: Собака и Кошка, которые реализуют метод идти, выводя в консоль "Собака идет" и "Кошка идет" соответственно.
Создатть Аннотацию @Autocreate которая служит для автоматического создания объектов классов, помеченных этой аннотацией.
Создать Класс AnimalPostProcessor, который сканирует пакет "src.main.java.lec7" на наличие классов, помеченных аннотацией @Autocreate, создает их объекты и вызывает метод идти у каждого созданного объекта.
Создать метод main и при старте вызвать метод из AnimalPostProcessor.
Каждый класс помеченный аннотацией @Autocreate должен быть создан и его метод идти должен быть вызван, выводя соответствующее сообщение в консоль.
